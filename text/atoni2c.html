Arduino IDE with NodeMCU I2C I2C introduction 

A serial bus interface connection protocol is known as I2C (Inter-Integrated Circuit). Given that it only requires two wires for communication, it is also known as a TWI (two-wire interface). SDA (serial data) and SCL (serial clock) are the names of those two wires. I2C is an acknowledgement-based communication protocol, meaning that after transferring data, the transmitter looks for an acknowledgment from the receiver to determine whether the data was successfully received by the receiver. I2C operates in two modes: Superior mode Master mode Data transfer between master and slave devices takes place through a serial data arc (SDA) link. The synchronous clock between master and slave devices is implemented via SCL (serial clock). Communication with a slave device is started by the master device. The master gadget demands

NodeMCU I2C Code using Arduino IDE (Master Device) 

#include <Wire.h> void setup() { Serial.begin(9600); /* begin serial for debug */ Wire.begin(D1, D2); /* join i2c bus with SDA=D1 and SCL=D2 of NodeMCU */ } void loop() { Wire.beginTransmission(8); /* begin with device address 8 */ Wire.write("Hello Arduino"); /* sends hello string */ Wire.endTransmission(); /* stop transmitting */ Wire.requestFrom(8, 13); /* request & read data of size 13 from slave */ while(Wire.available()){ char c = Wire.read(); Serial.print(c); } Serial.println(); delay(1000); }

#include <Wire.h> void setup() { Wire.begin(8); /* join i2c bus with address 8 */ Wire.onReceive(receiveEvent); /* register receive event */ Wire.onRequest(requestEvent); /* register request event */ Serial.begin(9600); /* start serial for debug */ } void loop() { delay(100); } // function that executes whenever data is received from master void receiveEvent(int howMany) { while (0 <Wire.available()) { char c = Wire.read(); /* receive byte as a character */ Serial.print(c); /* print the character */ } Serial.println(); /* to newline */ } // function that executes whenever data is requested from master void requestEvent() { Wire.write("Hello NodeMCU"); /*send string on request */
